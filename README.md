# My First Website

This repository contains a simple starter static website (index.html, styles.css, script.js).

How to publish with GitHub Pages:
1. In your repository, go to Settings → Pages.
2. Under "Build and deployment", select the branch (usually `main`) and the folder `/ (root)` then Save.
3. After a minute your site will be available at `https://<your-username>.github.io/myfirstwebsite/` (or a custom domain if configured).

Customize the site by editing index.html, styles.css, and script.js. If you'd like, tell me what colors, content, or sections you want and I can update the site for you.
